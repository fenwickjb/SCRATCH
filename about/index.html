    This Scratch Project Analyzer was created and intended to be used by Educators and Students alike to help them better understand and utilize the principles of Computer Science through SCRATCH. Created by Joseph O'Neill at Appalachian State University, this website is part of his Master's Thesis. To find and install this website on a seperate webserver, simply follow the link at the bottom of the page.

    To use this service, simply go to the Home page and input a SCRATCH project URL (Ex: https://scratch.mit.edu/projects/160539421/) or project number (Ex: 160539421). Once submitted, your results will be presented in a new webpage.
